Refactor the BLOG section UI to look modern, professional, and similar to high-quality SaaS websites.

Requirements:

1. Layout:

* Display blog posts in a responsive grid (3 columns on desktop, 1 column on mobile)
* Each blog should be a card with:

  * Image at the top
  * Title
  * Short description
  * "Read More" link

2. Styling:

* Use clean spacing and padding
* Add rounded corners (border-radius: 12px or more)
* Add soft shadow (box-shadow)
* Add hover effect:

  * Slight lift (transform: translateY(-5px))
  * Shadow increase
* Use consistent typography and spacing

3. Blog Card Content:
   Create 3 cards for:

* Best Time to Exchange Dollars in Uganda
* Why USD/UGX Exchange Rates Keep Changing
* How Real-Time Forex Alerts Can Save You Money

4. Images:

* Add relevant images for each blog:

  * Forex charts / graphs for exchange rate blog
  * Currency / money visuals
  * Data or analytics themed visuals
* Use placeholder images from:
  https://source.unsplash.com

Example:

* https://source.unsplash.com/600x400/?finance
* https://source.unsplash.com/600x400/?currency
* https://source.unsplash.com/600x400/?data,analytics

5. Code Structure:

* Use semantic HTML
* Create a container div for blog grid
* Each blog card should be reusable

6. Responsiveness:

* Mobile: 1 column
* Tablet: 2 columns
* Desktop: 3 columns

7. Keep It Simple:

* Use plain HTML, CSS, and minimal JS
* No frameworks

Goal:
Make the blog section look like a professional SaaS website, clean and visually appealing, not like a basic HTML page.
